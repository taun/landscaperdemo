Drawing Rules

Every fractal is defined by it’s drawing rules. A drawing rule tells FractalScapes how to draw the fractal on the the screen. Some sample rules are:

Draw a straight line in the current line color and thickness.
Move in a straight line but don’t draw the line.
Turn counter clockwise before drawing again.
Turn clockwise before drawing again.

A description of all the rules is available using the in-app help. There is help text with the rule description available by tapping on the rule. There is also a complete listing of the rules in the app Help tab.

The following screenshot shows the Starting rules (level 0) for the Sierpinski Triangle fractal. You can see it is fairly simple. A few straight lines and turns.

The yellow triangle in the following image is the result of drawing just the Starting rules with no replacements. This is detail level 0. 

The Sierpinski Triangle rules includes one replacement rule. Any occurrences of the placeholder (1) gets replaced by the rules to the right of the =>. These are rules very similar to the Starting rules but notice there are 2 rotations before the drawing starts and many additional (1) placeholders to be replaced in the next level of detail (iteration).

This image shows the result of the first level of replacements. The (1) in the first triangle image has been replaced with all of the rules in the => section which draws a smaller rotated triangle inside the first. Note that the new triangle has 3 new (1) placeholders. The next level of detail (level 2) will replace all of the new (1)s with new smaller triangles.

The following is the Sierpinski Triangle after 5 replacements (level 5). Pretty cool. In theory, the level of detail is infinite. In FractalScapes, you are limited to a maximum level based on memory usage and speed. Once it takes too long to draw a fractal, FractalScapes which not let you go to a higher level. So the maximum level of detail is different for each fractal and depends on the rules.
